cd folder path
Navigate to the folder

git init
Initialisation of the repo

git add file1.txt(filename)
file is added to the repo and now will be tracked

git commit -m "saving first file"
Saving the work in the file

git remote add origin https://github.com/NikJamdade/GitCommands.git(url from the github screen(browser))
Informing local git where to exactly upload the file

git remote show origin
Not mandatory but this helps to know where your changes are being pushed

git push origin master
Here we are uploading the changes to the remote master

git pull origin master
Here we're pulling the changes made directly on the repo

git clone https://github.com/erpoexample(repo url)
Here we're cloning the repo to our local system
